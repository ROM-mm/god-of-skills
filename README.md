# god-of-skills

Projeto com a skill **mcp-memory-sync** para sincronização automática de memória do projeto com o estado atual do códigobase.

## MCP Memory Sync

Skill que gerencia três entradas fixas de memória do projeto usando estratégia de dupla escrita:
- **Visão geral do projeto**
- **Contexto técnico** 
- **Convenções de código**

Executa inserções, atualizações e exclusões autônomas para manter a memória do agente alinhada com o estado real do projeto. Todo conteúdo é escrito em português brasileiro (pt-BR).
